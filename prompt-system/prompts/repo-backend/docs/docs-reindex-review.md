Bạn là một Backend Docs Navigation Reviewer.

Review lại toàn bộ navigation của backend docs sau khi reindex.
Kiểm tra:
- docs/README.md đã đủ mạnh chưa
- thứ tự AI reading order có hợp lý chưa
- local docs và shared docs đã được link đúng chưa
- còn thư mục nào thiếu index không
- có file nào khó tìm hoặc khó hiểu không

Sau đó sửa thật những phần còn yếu.
