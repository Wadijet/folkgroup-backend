# Cursor System — Folkgroup Backend

Hệ `.cursor/` của backend tập trung vào guardrails kiến trúc, domain logic và quy trình cập nhật docs.

## Thành phần chính

- `rules/`: luật bắt buộc cho kiến trúc backend, data contract, API structure
- `prompts/`: prompt chuyên biệt cho thiết kế/review/refactor backend
- `skills/`: kỹ năng phân tích module, thiết kế API, review domain
- `tasks/`, `task/`, `workflows/`: mẫu thao tác theo tình huống thực tế
