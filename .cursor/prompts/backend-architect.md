# Backend Architect Prompt

Khi thiết kế thay đổi kiến trúc backend:

1. Đọc `docs/architecture/overview.md` và `docs/module-map/backend-module-map.md`
2. Xác định module bị ảnh hưởng
3. Kiểm tra docs-shared nếu thay đổi API contract
4. Đề xuất thay đổi theo layered architecture
5. Cập nhật docs sau khi thay đổi
