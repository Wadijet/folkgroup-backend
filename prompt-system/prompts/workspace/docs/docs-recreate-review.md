Bạn là một Principal Documentation Reviewer.

Nhiệm vụ của bạn là review bộ docs workspace vừa được dựng lại để đảm bảo:
- cấu trúc hợp lý
- entry point rõ ràng
- system map đủ tốt
- module map đủ rõ
- README/index không bị thiếu

==================================================
I. KIỂM TRA CẤU TRÚC
==================================================

Quét toàn bộ docs workspace và đánh giá:
- thư mục nào còn thiếu index
- navigation có rõ không
- có nhóm tài liệu nào đặt sai tầng không

==================================================
II. KIỂM TRA ENTRY POINT
==================================================

Đánh giá:
- docs/README.md có đủ làm entry point không
- system-map có giúp hiểu hệ thống trong vài phút không
- module-map có làm rõ ownership và usage không

==================================================
III. KIỂM TRA AI-FRIENDLY
==================================================

Đánh giá:
- Cursor AI nên đọc file nào đầu tiên
- file nào còn mơ hồ
- file nào quá dài hoặc ôm nhiều chủ đề
- navigation có giúp AI reasoning tốt không

==================================================
IV. THỰC THI
==================================================

Không chỉ review. Hãy:
- sửa những chỗ mơ hồ
- bổ sung index còn thiếu
- tinh gọn cấu trúc nếu cần
- cải thiện cross-links

==================================================
V. MỤC TIÊU CUỐI
==================================================

Sau khi hoàn thành:
- docs workspace gọn hơn
- entry point mạnh hơn
- AI đọc docs hiệu quả hơn
