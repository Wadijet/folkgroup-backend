# Workflow: Review Backend Change

1. **Handler** — Mỏng? Chỉ parse/validate/service/response?
2. **Service** — Business logic đúng chỗ?
3. **Response** — Format chuẩn (code, message, data, status)?
4. **Organization** — OwnerOrganizationID, filter org?
5. **Errors** — Xử lý đầy đủ, Tiếng Việt?
6. **Docs** — Cập nhật nếu cần?
